# Manual de Identidad Visual: Uniformes Tamys (Web & App)

Este documento establece los parámetros visuales para el desarrollo de la interfaz de Uniformes Tamys. La identidad debe proyectar limpieza, confianza corporativa y calidad textil.

## 1. Colores Institucionales
La paleta se deriva de los tres colores presentes en el isotipo del cierre (zipper).

* **Color Primario:** `#0000FF` (Azul Eléctrico). Uso en la estructura general de la app y navegación.
* **Color de Acento:** `#EE1D23` (Rojo Vibrante). Uso exclusivo para Call to Action (CTA), alertas, promociones y elementos que requieran atención inmediata.
* **Color de Detalle:** `#FFD600` (Amarillo Cierre). Uso mínimo para acentos decorativos o etiquetas de categorías especiales.
* **Fondo (Background):** `#FFFFFF` (Blanco puro) para dar sensación de higiene y orden.

## 2. Tipografía Sugerida
Para equilibrar el estilo sans-serif del logo:
* **Títulos:** `Montserrat` (Semi-Bold). Moderna y profesional.
* **Cuerpo:** `Open Sans`. Amigable y con excelente legibilidad para catálogos de tallas y materiales.

## 3. Estilo de Componentes UI
* **Formas:** Bordes con radio de **10px a 12px**. Esquinas más redondeadas que sugieren suavidad (textil) y accesibilidad.
* **Iconografía:** Estilo "Lineal" (Thin/Light) para contrastar con lo cargado del logotipo del cierre.
* **Tratamiento de Imagen:** Uso de fotografías con modelos o texturas de tela, evitando sombras duras para mantener la limpieza visual.
